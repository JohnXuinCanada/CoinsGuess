# 🎮 双人猜硬币对决游戏

一个基于 HTML5 的双人猜硬币对战游戏，支持手机和电脑浏览器。

## 🎯 游戏介绍

这是一个策略性的猜测游戏，两位玩家通过猜测双方手中筹码总数来赢得对手的筹码。

### 游戏特色
- 🎨 精美的暗金主题界面
- 📱 完美支持手机触摸操作
- 🎲 公平的骰子先后顺序机制
- 🤖 智能 AI 对手（单机版）
- ✨ 流畅的动画效果

## 📖 游戏规则

1. **初始筹码**：每位玩家开始时有 10 枚筹码
2. **掷骰子**：决定谁先猜测
3. **出拳**：双方各自选择 1-3 枚筹码握在手中
4. **猜测顺序**：
   - 先猜者猜对 → 赢得所有筹码
   - 先猜者猜错 → 后猜者有机会猜测
   - 双方都猜错 → 筹码归还，重新开始
5. **获胜条件**：对手筹码耗尽或选择退出

### ⚡ 重要规则
- 第一个人猜错时，**不会透露实际总数**，保持游戏公平性
- 只有双方都猜错时才会公布答案
- 游戏进行中不能退出，只能在回合结束后选择退出
